# IRCTC App – Usability Audit & Redesign
Task 1 – UI/UX Internship

This project focuses on redesigning the IRCTC mobile application to improve
usability, clarity, and user experience.

-> Problems in Existing App
- Cluttered interface
- Confusing navigation
- Poor readability
- Complex ticket booking flow

-> Redesign Solution
- Clean and minimal home screen
- Clear call-to-action buttons
- Simple booking process
- Clear booking confirmation screen
-> Screens Designed
- Home Screen
- Ticket Booking Screen
- Confirmation Screen
